Mobile Health Units Interactive Story Map

A data-driven interactive web experience visualizing the geography, accessibility, and impact of Mobile Health Units (MHUs) across Texas. This project combines spatial storytelling, responsive web design, and interactive mapping to communicate how mobile healthcare infrastructure supports underserved communities throughout the state.

Built for Healthy Futures of Texas
 and published through Texas Women’s Healthcare Coalition
.

Project Overview

This project was designed as an immersive scrollytelling experience that integrates:

Interactive geographic visualization
Narrative-driven data presentation
Responsive design for desktop and mobile
Spatial storytelling techniques
Embedded methodology and documentation
Animated transitions and layered visual states

The experience guides users through:

The geography of healthcare access in Texas
Distribution of Mobile Health Units
Regional disparities and service gaps
Infrastructure and operational context
Supporting documentation and methodology

Design Goals

The project was developed to:

Make healthcare geography more understandable to non-technical audiences
Translate spatial inequities into accessible visual narratives
Support public health communication and advocacy
Create an engaging, modern storytelling experience for policy and community stakeholders
